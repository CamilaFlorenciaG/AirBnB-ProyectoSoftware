Para hacer funcionar este programa antes se debe 
1) tener instalado XAMPP y tener prendido Apache y MySQL (O algún programa similar)  
2) crear una base de datos llamada 'airbnb' en MySQL, o dentro de phpMyAdmin
3) Ejecutar PRIMERO el script SQL de personas y luego el script de alquileres.
y por ultimo ir a los archivos del programa "db.php", "conexión.php" en estos en las primeras líneas se tiene que poner en donde dice "user" y "pass" el usuario y la contraseña de sus cuentas de MySQL (normalmente suele ser user = root y contraseña = 123)
